Formula Reference
Breakeven
Fixed Costs ÷ (Price − Variable Cost per Unit)
Where: Price − Variable Cost = Contribution Margin per unit
Ex: $1M ÷ ($100 − $40) = 16,667 units to break even
Expected Value
EV = Σ P(outcome) × V(outcome)
Sum all scenarios. Probabilities must add to 100%.
Ex: 30% × $5M + 70% × $500K = $1.85M expected value
Market Sizing
Population → Segment % → Penetration % → Value per unit
State each assumption out loud before using it.
Ex: 260M adults × 65% credit-eligible × 20% travel affinity × $4K spend × 2% interchange ≈ $2.7B TAM
Weighted Average
Weighted Avg = Σ (Weighti × Valuei)
Always verify weights sum to 100% first.
Ex: 20% × $1,000 + 50% × $400 + 30% × $100 = $430 avg
Fixed vs. Variable Costs
Total Cost = Fixed + (Variable Rate × Volume)
Fixed: stays constant (salaries, licenses, rent)
Variable: scales with volume (per-unit processing, support)
Crossover point: FixedA ÷ (VarB − VarA)
Ex: $1M fixed ÷ ($30 − $15) variable diff = 66,667 unit crossover
